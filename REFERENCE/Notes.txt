Submission Checklist
--
  1. remove irrelevant/unnecessary console.log 
  2. check for linting errors 
      ng lint 
  3. generate code coverage report
      ng test --code-coverage 
  4. take a build 
      ng build 

  5. ignore node_modules, .angular, .git, .vscode 
  6. and then, zip the project 
  7. upload the project
